## 猛兽派对 自动钓鱼脚本
Party Animals 猛兽派对游戏自动钓鱼辅助程序

基于 Python + OpenCV + PyAutoGUI 开发，纯图像识别实现自动化操作

## 实现功能
1. **全自动钓鱼流程**：
自动抛竿 → 上鱼检测 → 智能拉鱼 → 自动收鱼，全流程无人值守
2. **异常自动检测**：
抛竿失败自动重试、鱼饵不足/鱼桶满载自动暂停脚本
3. **传奇鱼专属功能**：
检测到传奇鱼自动截图保存，自动创建截图文件夹
4. **防检测模拟**：
全随机时间算法（抛竿/拉鱼/间隔），完全模拟人工操作
5. **双配置切换**：
支持轻杆、重杆两套独立参数配置，一键切换
6. **自定义参数**：
通过配置文件自由修改所有操作时间、暂停快捷键
7. **便捷控制**：
全局快捷键一键暂停/继续脚本，支持自定义快捷键
8. **窗口校验**：
自动检测游戏窗口，确保脚本仅在游戏内运行
9. 尝试适配多分辨率支持（测试中）


## 使用方法（源码运行版）
1. 安装Python环境
2. 安装所需依赖
`pip install pyautogui opencv-python numpy keyboard psutil`
3. 打开main.py 并运行
`python main.py`

## 懒人版使用方法
直接下载Releases（发行版）内的 “盒意味.exe” 文件，双击运行后根据提示操作

## 运行要求
1. 游戏分辨率：1920×1080（全屏 / 窗口化均可）
2. 游戏状态：手持鱼竿，站在水边即可启动脚本
3. 操作规范：脚本运行时请勿操作鼠标左右键，请始终保持游戏窗口在最上层
